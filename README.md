# Pico Warriors


Short description :

Introducing Pico Warriors, a lost-and-found prototype app designed for university staff and students to effortlessly locate their lost items, fostering a connected campus community

Installation / setup instructions ;

1. Clone the repository
2. Open in your Java IDE
3. Run Main.java or visual studio code

usage intructions:

### 1. Access the Application
Open the live app prototype  here: https://preview--tag-back-buddy.lovable.app/

### 2. Register or Log In
- **New Users**: Click "Sign Up" to create an account
- **Existing Users**: Click "Log In" and enter your credentials

### 3. Tagging Items
- Go to the "Tag Item" section
- Enter item details and upload a photo
- Click "Generate Tag" to get a unique identifier
- Attach the tag to your item

### 4. Reporting Lost Items
- Go to the "Lost Items" section
- Enter the tag or description of the lost item
- Submit the report

### 5. Searching for Items
- Go to "Search Lost Items"
- Enter the QR code or scan  
- Browse the results to find matching items
